- How tto build

Solution is built using Visual Studio 2017
Open SimpleBilling.sln and run F7 (Build Solution)

- How to run
There are 3 files, used as input data:

1. Subscribers.xml - stores information about all the subscribers. Subscriber links Tarif plan from the Tariffs.xml by name.
2. Tariffs.xml - sores parameters for the biling calculator (fee, cost...)
3. CallsList.xml - stores list of call to be calculated. Links subscribers from Subscribers.xml

You can add new subscribers, tarif plan values, call parameters.
